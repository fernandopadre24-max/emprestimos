# **App Name**: CredSimples

## Core Features:

- Loan Application: Allows users to fill out a loan application form with personal and financial details.
- Interest Calculation: Calculate long-term loan interest based on principal amount, interest rate, and loan term.
- Customer List: Maintain a list of customers with their loan details, payment history, and contact information using a Cloud SQL database.
- Payment Tracking: Track payments made by customers, update loan balances, and generate payment schedules.
- Report Generation: Generative AI tool for producing custom reports that display all data associated with loan distributions, payment tracking, and overall profitability metrics.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and security in financial transactions.
- Background color: Light gray (#F0F2F5), a desaturated hue similar to the primary color, but very light to ensure legibility.
- Accent color: Soft green (#8BC34A) to highlight calls to action and successful transactions, creating a sense of growth and positivity.
- Headline font: 'Space Grotesk', sans-serif, used with 'Inter' for body
- Use clean, modern icons to represent loan types, payment options, and customer profiles.
- A clean, organized layout with clear sections for loan applications, customer lists, and payment tracking.
- Subtle animations for loading indicators and payment confirmations to enhance user experience.
